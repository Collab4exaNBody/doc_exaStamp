# **SNAP potential**
  
The examples related to theSNAP potential are located here:

```
cd ${XNB_SRC_DIR}/contribs/microStamp/samples/snap
```

This potential is computationally expensive to to its high complexity emergin from its core function, the BSO4 bispectrum descriptor. This potential is also a good starting point to benchmark applications.
