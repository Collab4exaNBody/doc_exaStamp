# **Spatial Regions**

This section covers how to carve out named spatial regions within the simulation domain, use them to restrict particle generation, and reassign particle IDs by region membership.

- [**Defining regions**](geometric.md) — the `particle_regions` operator, quadric-based shapes (boxes, planes, cylinders, spheres, cones), and particle-id ranges
- [**Restricting particle generation**](restricting.md) — grid masks, boolean region expressions, and user-defined source functions
- [**Tracking particles in regions**](tracking.md) — reassigning particle IDs by region membership
